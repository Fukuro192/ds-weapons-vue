# ds-weapons-vue

The Vue frontend of [ds-weapons-api](https://github.com/Fukuro192/ds-weapons-api).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
